# Proyecto JavaDrive

## ¿De qué trata el proyecto?
Es una aplicación en Java que sirve para gestionar el alquiler y las reservas de coches. 

## Reglas para trabajar en equipo
Para trabajar bien en equipo hemos seguido las siguientes normas:
- Cada vez que subamos algo a github, a cada commit le pondremos un mensaje claro de los cambios que hemos hecho.
- Intenamos siempre tener el código limpio y que los nombres de los archivos y de las clases sean lo más claro y entendible posible.
- Siempre antes de empezar a editar el código, haremos un pull para obtener los cambios más nuevos que hayan subido los demás compañeros para no tener errores.
